## Prerequisites to run locally

install nodejs  

e.g.  
```terminal
winget install nodejs
```

## Usage  

1. In the project folder  

```terminal
npm install
```  

2. Start the dev server  

```terminal
npx vite
```  

3. Open the url the server starts on
